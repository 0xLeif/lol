# lol

## Install

`v install zmeriksen.lol`

## Example:
```
import zmeriksen.lol as Lol

fn main() {
	lol := Lol.Lol{
		Lol.Config{
			Lol.Style.normal
		}
	}
	lol.print("Hello V World")
}
```
